# Prompt - Refatoracao

Use este prompt para refatoracoes planejadas.

## Tarefa

Refatorar sem mudar comportamento funcional.

## Regras

- Mapear comportamento atual antes de editar.
- Manter colecoes e campos Firestore.
- Manter compatibilidade com sessoes e dados existentes.
- Evitar refatoracao visual sem pedido explicito.
- Rodar testes antes e depois quando possivel.
- Documentar motivo, escopo e risco.
- Se a refatoracao criar padrao duradouro, registrar ADR.
