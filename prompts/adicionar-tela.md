# Prompt - Adicionar Tela

Use este prompt quando precisar criar uma nova tela.

## Tarefa

Adicionar tela ao app CUIABAMT seguindo padroes visuais, autenticacao e permissoes existentes.

## Regras

- Ler `docs/UI.md`, `docs/SECURITY.md`, `docs/WORKFLOW.md` e `AGENTS.md`.
- Reaproveitar cabecalho, botoes e padroes existentes.
- Validar sessao local e aguardar Firebase Auth.
- Bloquear gravacao para perfil CONSULTA quando aplicavel.
- Nao criar novas colecoes sem documentar em `docs/COLLECTIONS.md`.
- Testar em mobile e desktop.
- Atualizar documentacao da nova tela.
