# Prompt - Corrigir Bug

Use este prompt para corrigir defeitos em producao.

## Tarefa

Diagnosticar, corrigir, testar e documentar um bug.

## Regras

- Confirmar que o repo e `CuiabaMT`.
- Reproduzir ou entender o erro antes de editar.
- Verificar `git status --short`.
- Fazer mudanca minima e segura.
- Nao alterar regras de negocio sem justificar.
- Nao executar deploy se o pedido for apenas analise.
- Rodar testes relevantes e registrar o resultado.
- Atualizar `docs/BUGS.md` ou `docs/CHANGELOG.md` se o bug for relevante.
